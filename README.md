API REST con ASP.NET Core 8 para gestión de usuarios y tareas. Requisitos: .NET 8 SDK, SQL Server (local o Docker), y paquetes NuGet: Microsoft.EntityFrameworkCore.SqlServer, Microsoft.EntityFrameworkCore.Tools, Swashbuckle.AspNetCore. Instalación: 1) Clonar repositorio, 2) dotnet restore para instalar paquetes, 3) Configurar cadena conexión en appsettings.json, 4) dotnet run para desarrollo o docker-compose up para contenedores. Endpoints: /api/Usuario (CRUD usuarios) y /api/Tarea (CRUD tareas con asignación a usuarios). Documentación: Swagger en /swagger. Base de datos: SQL Server con Entity Framework Core, auto-creación con migraciones. Docker: Incluye Dockerfile y docker-compose.yml con SQL Server integrado.

⚠ Nota sobre .NET 10: Actualmente se encuentra solo en versión Preview con numerosos bugs e inestabilidad, por lo que NO es recomendable para proyectos de producción. Este proyecto fue desarrollado en .NET 8 (LTS), la versión estable más reciente con soporte a largo plazo garantizado hasta noviembre de 2026.
